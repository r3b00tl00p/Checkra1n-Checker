# Checkra1n-Checker
This python script let you know if Checkra1n is released or no

## System Requirement
You need a mac to run this script

## Installation
You need to install the latest version of Python from here : https://www.python.org/ftp/python/3.8.0/python-3.8.0-macosx10.9.pkg

Then install pip and required files: in terminal type
```bash
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
sudo python get-pip.py
sudo pip install requests
```

Now you can run the script by typing

```bash
python
```
and drop the CHECKra1n.py file on terminal and hit enter
